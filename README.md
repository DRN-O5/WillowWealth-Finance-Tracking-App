<h1>Finance Tracking Application</h1>
<h2>Overview</h2>

A finance tracking web application that helps users record, analyze, and visualize financial data. The project focuses on structured data storage, insights through charts, and notification support.

<h2>Libraries & Frameworks Used</h2>

Next.js – Framework for optimized, server-rendered web applications.

MySQL – Relational database for persistent financial data.

Recharts – Visualizes financial trends using interactive charts.

Twilio – Sends notifications and alerts related to financial activity.
